== Description ==

**Real-Time Feedback plugin automatically track common usability behaviors that hinder or irritate users.
Take actions in real-time with a built-in feedback poll to understand how visitors are really using your website.**

##### Real-Time Feedback Plugin Detects:
* **Rage Clicks** - Detect multiple clicks or taps, rapidly, in the same area. Find when your website doesn't react the way your users thought it should.
* **Frustration** - Detect behaviors where users encounter indicators like chaotic mouse movement, random clicks or multiple up and down page scrolls.
* **Wasted Clicks** - Detect misleading links that force users to bounce back and forth between a routing page and subpages linked from it.

== Installation ==
1. Unzip and upload the Real-Time Feedback plugin to your /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure your desired settings for the feedback poll
4. Done! You're good to go!
